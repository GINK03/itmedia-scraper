　2014年9月3日（現地時間）、韓国Samsung Electronicsは独ベルリンで開催する「IFA 2014」に先駆けて新製品発表会を行い、「GALAXY Note 4」「GALAXY Note Edge」「Gear S」「Gear VR」の4製品を発表した。 　「Samsung Unpacked 2014 Episode 2」として開催された新製品発表会には、Samsungのセールス・ITマーケティング・モバイルコミュニケーション部門の室長であるD.J.リー氏が登壇した。同氏はまずS「amsungはGALAXY Sシリーズでスマートフォン市場に参入以来、スマートフォンのトレンドを作り上げてきた」と説明。例えば2011年に発売したGALAXY Noteは大画面スマートフォン＝ファブレットという分野を新たに開拓し、また2013年に投入したGALAXY Gearはウェアラブルデバイスの先駆け的な位置付けの製品になっている。 　Samsungの考えるスマートフォンとは、人々のコミュニケーションをより密接にし、日々の生活をより豊かなものにする製品だという。スマートフォンで数々のトレンドを作り上げてきた同社は「これからも開発の手を緩めることはない」（リー氏）という。そして現時点の同社の技術の集大成ともいえる製品として、「GALAXY Note 4」と「GALAXY Note Edge」を発表した。 　両モデルとも付属のスタイラスペンの機能を強化し、Webページや写真を簡単にクリップして友人と共有できる。また、製品そのものもよりプレミアムなデザインとして高級感を持たせている。さらに、GALAXY Note Edgeは曲面ディスプレイを採用。本体右側は緩やかなカーブを描くディスプレイで覆われており、ここに情報を表示したり、タッチ操作によりコントローラーとして利用したりできる。今までのスマートフォンにはなかった新たなユーザー体験は、新しいライフスタイルを提供できるだろうと同氏は話した。 　そして新しいウェアラブルデバイスの「Gear S」「Gear VR」は身に付けることで生活を楽しく便利なものにしてくれるという。これらの新しいデバイスには新たなプレーヤーがコンテンツやサービスを提供することにもなり、スマートフォン市場全体のさらなる発展も期待できるだろう。 　続けてGALAXY Note 4の新機能などが説明された。これまでのNoteシリーズからの大きな変化は本体の高級化とスタイラスペン「Sペン」の機能強化だ。これにより、スマートフォンから「高級なデジタル文具」とも呼べそうな製品に仕上がっている。以下、強化ポイントや新機能などを説明する。 　5.7型の大きな画面でPCのようなマルチウィンドウが利用できる。さらにはSペンで自在に画面クリップや手書きができるなど、他社のスマートフォンにはない機能が多数備わっている。 　GALAXY Note 4では「デザイン」「大画面とマルチタスク」「Sペン」「カメラ」の4つのポイントが大きく強化された。それによりGALAXY Note 4は同社のフラッグシップにポジショニングされ、GALAXY S5と並ぶ最上位モデルとなった。 　2013年のモデル、GALAXY Note 3はプラスチック製なのにまるで革のような風合いの背面カバーで一躍有名になった。GALAXY Note4も引き続き同じような仕上げがされており、さらに高級感を増したデザインとなっている。この加工は見た目の良さだけではなく、手に持った時にすべりにくいという利点も持つ。 　本体のフレームは金属素材となり、見た目の高級感をさらに増している。そして耐久性も高まったことでよりハードな使い方にも耐えうる。なお純正品としてハードプロテクターケースも販売される。 　黒、白に加えブロンズゴールドとピンクの4色展開。過去のモデルではピンクがあとから出てくる、といったこともあったが、Note4では最初から全色がそろって発売になる。高解像度の有機ELディスプレイにより、写真だけではなく電子書籍の利用にも適している。 　2つのアプリを同時に利用できるマルチタスク機能も進化した。GALAXY Note 3では画面を上下に分割するだけだったが、Note4ではミニウィンドウにしてオーバーレイ表示することも可能になった。 　より本物のペンの書き味を実現するために、Sペンの筆圧感度は従来品より倍になった。Sペンを使いWebページや地図の一部画面を簡単にクリッピングしたり、文字をなぞるだけで選択したりできるようになった。PCのマウスのようにSペンを使うことができるようになっている。 　メインカメラは有効約1600万画素に、そして正面のインカメラも370万画素と強化。インカメラはF1.9と明るさも増している。最近流行の自分撮りのみならず、インカメラを使って集合写真も簡単に撮れるようになった。パノラマ撮影のようにカメラを動かすと120度の角度で写真撮影ができる。 　閉じた状態でも開いた窓から時計表示ができるフリップカバーが引き続き販売されるほか、LEDで各種表示ができるカバーも登場する。女性ユーザーをターゲットにした、スワロフスキーとコラボしたカバーも用意。電子文具として使いたいユーザーのために、モンブランとの提携も初めて行った。モンブランのペンと同じ作りのSペンが2種類と、フリップカバーも販売される。 　本体の右側面を覆うようにカーブしたディスプレイを採用するGALAXY Note Edge。基本的な機能とスペックはGALAXY Note4とほぼ同等だ。GALAXY Note Edgeはそれに加え、このカーブした部分のディスプレイ＝エッジ・ディスプレイ部分に様々な機能が加わっている。 　Sペンを収納する本体の右側が曲面ディスプレイとなっているGALAXY Note Edge。水が流れ落ちるような美しいデザインになっている。この曲面ディスプレイ部分は正面側とは別の表示ができるようになっており、今までのスマートフォンにはない新しい使い方ができる。 　新しいユーザーインタフェースとして、このエッジ・ディスプレイ部分にはアプリアイコンが標準で表示される。ワンタッチでよく使うアプリを起動することが可能だ。 　エッジ・ディスプレイ部分では専用アプリも利用できる。ボイスメモやストップウォッチは本体側のアプリを起動せずとも、側面部分からワンタッチで利用できる。また上記アプリは側面いっぱいまでディスプレイがあるNote4ならではのアプリだろう。 　エッジ・ディスプレイには天気予報やニュースを流すこともできる。そのためGALAXY Note Edge用のフリップカバーは正面部分の右側が短くなっており、この部分が常に見える状態になっている。 Copyright© 2017 ITmedia, Inc.
